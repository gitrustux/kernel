# Rustux Architecture

## Kernel
- **Memory Management**: Manages physical and virtual memory.
- **Process Management**: Handles processes and threads.
- **File System**: Implements ext4 for file I/O.

## User Space
- **Shell**: A simple command-line interface.
- **Text Editor**: A basic text editor.

## Networking
- **TCP/IP Stack**: Implements basic networking protocols.

## Security
- **Capability-Based Security**: Grants processes specific permissions.
